# Create pipelines that enable IP ranges

```yaml
jobs:
  unit_test:
    circleci_ip_ranges: true
    docker:
      - image: cimg/node:18.14.0
    steps:
      - checkout
      - node/install-packages
      - run: npm run test:ci
```

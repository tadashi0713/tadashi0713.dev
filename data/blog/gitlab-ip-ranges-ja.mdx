# セルフホスト版

# IP アドレス範囲機能を

この

```yaml
jobs:
  unit_test:
    circleci_ip_ranges: true
    docker:
      - image: cimg/node:18.14.0
    steps:
      - checkout
      - node/install-packages
      - run: npm run test:ci
```

# Docker ビルドを利用したい場合

今回は IP アドレス範囲を利用したソリューションになっています。

現在 IP アドレス範囲は Docker Executor のみ
